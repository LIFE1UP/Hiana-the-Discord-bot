# 디스코드 봇
## 봇을 활성화 시키기
먼저, bot.py에서 디스코드 API key를 **token** 변수에 넣어준다. $python bot.py를 입력한다. 
>라이브러리 설치가 되어있지 않다면 install_library.exe를 가장 먼저 실행시킨다.

## haina를 훈련시키기
'/txbs/textbook.json'에서 노드 구조로된 데이터를 이해하고 적절하게 바꾸면 된다. 그러고 $python train.p를 입력한다.
